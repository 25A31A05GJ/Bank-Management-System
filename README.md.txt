# Bank Management System (C++)

## 📌 Description
This project is a simple Bank Management System implemented using Linked List in C++.

## 🚀 Features
- Create Account
- Display Accounts
- Deposit Money
- Withdraw Money
- Delete Account

## 🧠 Data Structure Used
- Singly Linked List

## ⚙️ Time Complexity
All operations: O(n)

## 💾 Space Complexity
O(n)

## 🛠️ Technologies
- C++
